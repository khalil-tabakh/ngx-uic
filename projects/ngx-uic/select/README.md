# NgxSelectModule

The `NgxSelectModule` provides a fully customizable select component for Angular applications. It includes the `NgxSelectComponent` along with a set of directives for building accessible, single and multiple selection dropdowns using native HTML elements.

## Features

✅ Single & Multiple selection modes

🎯 Pointer & keyboard interaction

♿ WAI-ARIA compliant listbox pattern

🪟 Native `<dialog>` & HTML Popover API support

🎨 Fully customizable trigger, popup, and option templates

🔗 Automatic trigger–popup association

📍 Native CSS Anchor Positioning support

🧩 Works with arbitrary value types

📝 `ControlValueAccessor` & `FormValueControl` support
