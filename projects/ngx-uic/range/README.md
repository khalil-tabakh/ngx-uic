# NgxRangeModule

The `NgxRangeModule` provides a reusable range slider component for Angular applications. It includes the `NgxRangeComponent`, a highly customizable Angular component for creating single and double range-based sliders. It provides flexibility for setting minimum, maximum, and step values and emits events for value changes.

## Features

✅ Single & Double slider modes

🎯 Pointer-based interaction

🔢 Flexible value configuration

📍 Custom step values

🟢 Optional tick marks

🧩 Segmented track support

📐 Custom relative origin

👆 Hover tracking

🎨 Dynamic styling
