# NgxScrollerModule

The `NgxScrollerModule` provides a reusable scrollable container component for Angular applications. It includes the `NgxScrollerComponent`, a customizable Angular component for creating bi-directional infinite scroll containers, designed for large lists, grid layouts, and dynamic content containers.

## Features

⚡ Virtualization for large dynamically sized elements

🔍 IntersectionObserver-driven item visibility tracking

♻️ Windowing with automatic expansion/shrinking

↕️ Emits events when reaching the start or end of the list

📦 Supports horizontal and vertival flexbox, grid, and normal flow

🧭 Works with custom scroll containers & root margins

🔧 Controlled or automatic emission logic

🧱 Built using Angular signals
